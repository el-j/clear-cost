# Clear Cost

3D-print quoting app for material, energy, wear, and customer-ready pricing.

## Tech Stack

| Tool | Purpose |
| --- | --- |
| React | UI framework |
| TypeScript | Type safety |
| Vite | Build tool and dev server |
| Tailwind CSS v4 | Styling |
| React Router | Hash-routed pages |
| Zustand | Persisted project store |
| Vitest | Unit tests |
| Playwright | E2E tests |

## Notes

- The app is fully client-side and responsive on mobile.
- Current project metadata and the saved project library are stored locally in the browser via Zustand persistence.
- The Projects page is the place to save, load, duplicate, and delete project cards.
